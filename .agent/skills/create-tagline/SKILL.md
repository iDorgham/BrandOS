# SKILL.md

Generates creative taglines that capture a brand's essence.

## Parameters
- `brand`: The name of the brand.
- `values`: Key pillars or personality traits of the brand.

## Examples
### BrandOS Tagline
```bash
create-tagline --brand "BrandOS" --values "AI-powered, efficient, creative management"
```
