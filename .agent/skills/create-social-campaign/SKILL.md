# SKILL.md

Builds a platform-specific social strategy for a specific campaign.

## Parameters
- `campaign`: The name of the campaign.
- `goals`: Social-specific goals (e.g., "engagement", "reach").
- `audience`: Platform-specific audience segments.

## Examples
### Instagram Engagement Campaign
```bash
create-social-campaign --campaign "Spring Sale" --goals "increase comments and shares" --audience "design enthusiasts"
```
