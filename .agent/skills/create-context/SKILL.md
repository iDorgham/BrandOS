# SKILL.md

Boilerplate generator for React Context APIs to manage global state.

## Parameters
- `name`: The base name for the Context (e.g., "Auth").
- `state`: Description of the state variables and functions to expose.

## Examples
### Theme Context
```bash
create-context --name Theme --state "mode: 'light' | 'dark', toggleTheme"
```
