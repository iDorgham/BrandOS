# SKILL.md

Builds a high-level timeline for product evolution and feature delivery.

## Parameters
- `product`: Name of the product or service.
- `features`: List of key features to be prioritized.
- `timeline`: Expected delivery windows (e.g., Q1, Q2).

## Examples
### Yearly Roadmap
```bash
create-roadmap --product "Brand DNA" --features "collaboration, analytics, API" --timeline "Q3, Q4"
```
