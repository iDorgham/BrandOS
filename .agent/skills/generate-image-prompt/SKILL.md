# SKILL.md

Creates effective prompts for various AI image generation tools.

## Parameters
- `subject`: The primary subject of the image.
- `style`: The art style or aesthetic (e.g., minimalist, 3D render, oil painting).
- `details`: Specific details like lighting, color palette, or composition.

## Examples
### Minimalist Logo
```bash
generate-image-prompt --subject "logo design" --style "minimalist" --details "blue and white, clean lines"
```
