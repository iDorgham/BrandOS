# SKILL.md

Defines the configuration and persona for an AI agent.

## Parameters
- `purpose`: The core mission or goal of the agent.
- `capabilities`: The specific tools or actions the agent should be able to perform.

## Examples
### Support Agent
```bash
build-ai-agent --purpose "customer support" --capabilities "answer questions, escalate issues, check order status"
```
