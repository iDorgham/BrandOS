# SKILL.md

Writes various types of marketing and product copy with a specific tone.

## Parameters
- `type`: The format of the copy (e.g., headline, email, product description).
- `topic`: The subject or product being described.
- `tone`: The desired brand voice (e.g., professional, bold, friendly).

## Examples
### Product Description
```bash
write-copy --type "product-description" --topic "wireless noise-canceling headphones" --tone "professional but excited"
```
