# SKILL.md

Structures a full-scale marketing effort with measurable objectives.

## Parameters
- `name`: The internal name for the campaign.
- `goals`: Key performance indicators (KPIs) or objectives.
- `channels`: Media channels to be used (e.g., email, SEM, social).

## Examples
### Product Launch Campaign
```bash
plan-campaign --name "BrandOS V2 Launch" --goals "1000 new signups" --channels "social, email, influencer"
```
