# SKILL.md

Crafts high-quality prompts for LLMs like GPT-4, Claude 3, and Gemini.

## Parameters
- `task`: The specific objective to accomplish.
- `model`: The target LLM (GPT, Claude, Gemini, etc.).
- `context`: Background information or constraints to guide the model.

## Examples
### Marketing Copy Prompt
```bash
write-prompt --task "generate marketing copy" --model gpt-4 --context "brand guidelines for a tech startup"
```
