# SKILL.md

Creates detailed UI/UX specifications for a specific interface element.

## Parameters
- `component`: The name of the component to design.
- `functionality`: Description of what the component must do.
- `style`: Desired visual direction or design tokens.

## Examples
### Login Form Design
```bash
design-component --component "LoginForm" --functionality "email, password, keep me logged in, forgot password"
```
