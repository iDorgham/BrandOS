# SKILL.md

Identifies high-value search terms for content and SEO strategies.

## Parameters
- `topic`: The core subject area.
- `competitors`: Websites to analyze for keyword overlap.

## Examples
### Design Software Keywords
```bash
keyword-research --topic "AI design tools" --competitors "canva.com, simplified.com"
```
