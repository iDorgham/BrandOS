# SKILL.md

Detailed scheduling for social media post frequency and themes.

## Parameters
- `platforms`: List of social media platforms.
- `frequency`: Post frequency per week/day.
- `themes`: Core content themes for the posts.

## Examples
### Weekly LinkedIn Schedule
```bash
plan-social-content --platforms "LinkedIn, X" --frequency "3 times per week" --themes "industry news, team insights"
```
