# SKILL.md

Plans a schedule for publishing marketing content across channels.

## Parameters
- `duration`: The timeframe for the calendar (e.g., monthly, quarterly).
- `topics`: Core themes or specific topics to cover.

## Examples
### Monthly Social Calendar
```bash
create-content-calendar --duration "monthly" --topics "product launches, customer stories, team culture"
```
