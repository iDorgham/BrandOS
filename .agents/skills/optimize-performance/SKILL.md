# SKILL.md

Applies patterns to improve the application's speed and efficiency.

## Parameters
- `target`: What to optimize (e.g., "bundle", "database queries", "re-renders").
- `focus`: Specific area or component to prioritize.

## Examples
### Bundle Optimization
```bash
optimize-performance --target "bundle" --focus "tree-shaking and lazy-loading"
```
