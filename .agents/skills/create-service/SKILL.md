# SKILL.md

Generates a service class or module for handling specialized logic or API calls.

## Parameters
- `name`: Name of the service.
- `methods`: List of functions or capabilities the service should have.

## Examples
### User Service
```bash
create-service --name user --methods "getProfile, updateProfile, deleteAccount"
```
