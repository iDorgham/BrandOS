# SKILL.md

Handles the deployment process to cloud platforms.

## Parameters
- `platform`: The target platform (e.g., "vercel", "netlify").
- `env`: The target environment (e.g., "production", "preview").

## Examples
### Vercel Deployment
```bash
deploy --platform "vercel" --env "production"
```
