# SKILL.md

Generates persuasive copy and visual concepts for paid advertising.

## Parameters
- `type`: The platform or format (e.g., social, search, display).
- `product`: The item or service being advertised.
- `audience`: The target demographic or segment.

## Examples
### Social Ad Concept
```bash
create-ad --type "social" --product "automated branding tool" --audience "frequent startup founders"
```
