# SKILL.md

Generates a foundational design system document with specific visual tokens.

## Parameters
- `brand`: The name of the brand.
- `colors`: Primary, secondary, and accent colors (e.g., HEX codes or names).
- `typography`: Recommended font families for headings and body text.

## Examples
### Brand Style Guide
```bash
create-style-guide --brand "MyBrand" --colors "blue, white, dark-gray" --typography "Inter, Roboto Mono"
```
