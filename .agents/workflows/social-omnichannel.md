description: 30-day content calendar across all social platforms
---

1. **Social Audit**: Use `social-media-strategist` to audit existing presence and consistency.
2. **Define Audience**: Use `social-media-manager` to refine target segments for each platform.
3. **Develop Strategy**: Use `social-media-strategist` and `create-social-campaign` for a 30-day plan.
4. **Content Themes**: Define 4 weekly themes using `content-strategist` and `create-content-calendar`.
5. **Content Creation**: Batch create content for each of the 4 weeks using `copywriter`.
6. **Visual Assets**: Use `graphic-designer` and `generate-image-prompt` to create 60 unique visuals.
7. **Schedule Posts**: Use `social-media-manager` to organize the final content calendar.
8. **Setup Analytics**: Configure a dashboard to track engagement, reach, and conversions.
