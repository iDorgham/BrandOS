---
description: Create new AI agent with full configuration
---

1. **Define Agent Role**: Use `product-planner` to establish name, domain, and core mission.
2. **Define Capabilities**: Outline the specific actions and tools the agent requires.
3. **Identify Skills**: Map capabilities to existing or new required skills.
4. **Acquire Skills**: Reuse existing skills or use `prompt-engineer` and `build-ai-agent` to create new ones.
5. **Write System Prompt**: Use `prompt-engineer` and `write-prompt` to craft the agent's core instructions.
6. **Define Rules**: Establish operating guidelines and constraints.
7. **Create Configuration**: Assemble all components into a final agent definition.
8. **Register Agent**: Add the new agent to the workspace registry.
9. **Verify**: Use `qa-engineer` to test the agent with representative test cases.
