# OpenCode & Kilo Operational Rules

This rule defines how OpenCode and Kilo Code agents must operate in this workspace.

## Governance
These tools are bound by the **Brand DNA Governance Matrix** located in `.agents/config/governance-matrix.json`.

## Specification Adherence
Follow all specifications in `.agents/protocols/`:
- **Workflow DSL**: For complex feature additions.
- **Communication Protocol**: For cross-domain handoffs.
- **HITL Protocol**: For safe execution.

## Verification
Before completing any task, verify your output against the relevant `SKILL.md` definitions in `.agents/skills/`.
